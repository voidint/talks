### 本地渲染
- 下载本工程至本地
```shell
$ cd ${YOUR_WORKSPACE} && git clone https://github.com/voidint/talks.git
```
- 安装present
```shell
$ go get -u -v golang.org/x/tools/cmd/present
```
- 运行present
```shell
$ cd ${YOUR_WORKSPACE}/talks && present
```
